🔑 Пароль адмінки

💡 Якщо забув пароль або хтось украв доступ

Напиши Telegram-боту /reset_admin_password — він згенерує новий пароль і надішле в чат.

⚠️ Cloudflare KV має propagation до 60 секунд — старий пароль може ще працювати в інших регіонах ~1 хв. Через 60 сек точно не працює.

Для миттєвої зміни (з цієї ж локації, без затримки) — використовуй кнопку нижче ⬇️

🚨 Миттєва зміна пароля (з браузера)

Згенерує випадковий пароль 14 символів і одразу його застосує. Старі сесії у твоєму регіоні CF — невалідні миттєво. Пароль надсилається в Telegram + показується тут.

Після зміни — тебе викине на сторінку логіна, треба зайти з новим паролем. «Скинути» видаляє перевизначення в KV → знову працює пароль з Cloudflare Secrets.

🔐 Двофакторна авторизація (2FA)

Після правильного пароля надійде 6-значний код вибраним каналом. Без коду — увійти не можна.
Якщо хочеш окремий чат для кодів — введи його ID. Інакше — той самий, що для замовлень.
📧 Для email-2FA треба зареєструватись на resend.com (безкоштовно — 3000 листів/міс). Створіть API ключ → впишіть сюди. Можна слати з onboarding@resend.dev на свій email без верифікації домена.
⚠️ Перш ніж вмикати 2FA — тестовий код має дійти. Інакше залишишся без доступу.

🚨 Сповіщення про ліміти Cloudflare

Якщо вичерпано ліміт KV writes/lists/reads або R2 — приходить сповіщення з поясненням що зламалось і як полагодити.
Захист від спаму — щоб не приходило 1000 сповіщень підряд при тривалому виченні ліміту. За замовчуванням: 60 хвилин.

🔓 Сповіщення про вхід в адмінку

У повідомленні буде час, IP та браузер. Якщо хтось зайде з чужого IP — побачите одразу і зможете швидко змінити пароль.
Увімкнено: сесія живе тільки до закриття браузера. Безпечніше якщо комп'ютером користуються інші.
Вимкнено (за замовчуванням): сесія живе 30 днів. Зручніше — не треба вводити пароль кожен раз.

⚠️ Поради з безпеки

Токен бота дає повний контроль над ним. Якщо він витече — хтось зможе від імені бота надсилати повідомлення, отримувати апдейти, змінювати команди. Не діліться адмін-паролем ні з ким та не показуйте цей екран стороннім.

Якщо щось сталося — йдіть до BotFather → /revoke → створіть новий токен і впишіть сюди.
