📦 Нова Пошта —

ℹ️ Коли увімкнено — у клієнта в кошику автоматично зʼявляється пункт «Нова Пошта» першим у списку доставки + автозаповнення міст і відділень. Цей пункт не зникає навіть якщо ти видалиш його з нижнього списку «Способи доставки».
Як буде виглядати у клієнта в радіокнопці. За замовчуванням: «📦 Нова Пошта».
Отримати безкоштовно: Кабінет НП → Налаштування → Для розробників → «Створити API ключ» → скопіюй сюди.
Ключ видно лише в адмінці, у публічний API не передається.

☁️ Cloudflare Analytics API опціонально

ℹ️ Дозволяє бачити реальне використання KV/R2/Pages за сьогодні (читання, записи, трафік) прямо в нашій /admin/stats/. Без цього — тільки в Cloudflare Dashboard.
Як створити:
  1. Іди на dash.cloudflare.com/profile/api-tokens
  2. Натисни «Create Token»
  3. Знайди шаблон «Read analytics and read account analytics» → Use Template (або Custom Token)
  4. Permissions: Account → Account Analytics → Read
  5. Account Resources: Include → твій акаунт
  6. Continue → Create → скопіюй токен ОДРАЗУ (більше не покажеться)
Де взяти: Cloudflare Dashboard → справа внизу «Account ID» → Copy.
Заповнено за замовчуванням. Якщо змінив namespace — постав сюди свій. Знайти: Storage & Databases → KV → SHOP_KV → ID.
Опціонально, але дуже корисно. Дозволяє миттєво очищати кеш сайту при змінах в адмінці або через Telegram-бот (магазин включається/вимикається одразу, без 30-60 сек затримки).

Як знайти: Cloudflare Dashboard → клікни на свій домен (sotosprint.store) → справа внизу «Zone ID» → Copy.

Важливо: Твій API token також має мати permission Zone → Cache Purge → Edit. Якщо ні — додай. (My Profile → API Tokens → твій токен → Edit → Add permission).
⚠️ Токен зберігається тільки в нашому KV (як інші паролі). Не передається назовні. Якщо хочеш видалити — очисти поле і збережи.
Якщо увімкнено — у правому верхньому кутку /admin/ з'явиться компактний індикатор: KV writes/reads/lists з кольоровим прогресом і часом до скидання.
